# fandark

饭否阴暗面工具

## 使用方法

下载并安装依赖

```bash
$ git clone git@github.com:LitoMore/fandark.git
$ cd fandark
$ npm install
```

复制配置文件模版

```bash
$ cp config.json.example config.json
```

配置应用

```json
{
  "CONSUMER_KEY": "key",
  "CONSUMER_SECRET": "secret",
  "FANFOU_USERNAME": "username",
  "FANFOU_PASSWORD": "password"
}

```

运行

```bash
$ node index.js
```
