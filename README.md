# fandark

Fanfou-Dark Toolkit

## Usage

Download & Install dependencies

```bash
$ git clone git@github.com:LitoMore/fandark.git
$ cd fandark
$ npm install
```

Prepare config file

```bash
$ cp config.json.example config.json
```

Configuration

```json
{
  "CONSUMER_KEY": "key",
  "CONSUMER_SECRET": "secret",
  "FANFOU_USERNAME": "username",
  "FANFOU_PASSWORD": "password",
  "REQUEST_LIMIT": 10
}

```

Run

```bash
$ node app.js
```
